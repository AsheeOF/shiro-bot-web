Política de Privacidad

Última actualización: Enero 2025

1. Qué datos recopilamos

Shiro Bot recopila únicamente los datos mínimos necesarios para su funcionamiento:

DatoPor quéDónde se guarda
ID de usuario de DiscordIdentificar al jugador en el sistema de XPServidor local
ID del servidorSeparar estadísticas entre servidoresServidor local
Nombre de usuarioMostrar en el leaderboardServidor local
Cantidad de mensajes enviadosCalcular XP y rangoServidor local
XP acumuladoSistema de rangosServidor local

No recopilamos el contenido de tus mensajes, información personal, correos electrónicos ni datos financieros.

2. Cómo usamos los datos

Los datos recopilados se usan exclusivamente para:

  • Calcular y mostrar tu rango y XP en el servidor
  • Generar el leaderboard del servidor con el comando !top
  • Notificar subidas de rango en el canal

3. Compartir datos

No compartimos, vendemos ni transferimos tus datos a terceros bajo ninguna circunstancia. Los datos permanecen en el servidor donde está alojado el bot.

4. Retención de datos

Los datos se conservan mientras el bot esté activo en el servidor. Si el bot es expulsado del servidor, los datos asociados a ese servidor pueden eliminarse. Puedes solicitar la eliminación de tus datos en cualquier momento.

5. Servicios de terceros

Shiro Bot se integra con los siguientes servicios externos:

  • Discord API — Para funcionar como bot. Consulta la política de privacidad de Discord.
  • Jikan API (MyAnimeList) — Para obtener horarios de anime. Sin datos personales enviados.
  • Anthropic API — Para el comando !anime. Solo se envía tu consulta de texto, sin datos de identidad.

6. Tus derechos

Tienes derecho a:

  • Solicitar qué datos tenemos sobre ti
  • Pedir la eliminación de tus datos con el comando !stats --reset
  • Contactarnos para cualquier duda sobre privacidad

7. Seguridad

Los datos se almacenan localmente en el servidor que hostea el bot. No garantizamos seguridad absoluta, pero tomamos medidas razonables para proteger la información.

8. Menores de edad

Shiro Bot no está dirigido a menores de 13 años. Si eres menor, necesitas el consentimiento de un adulto para usar Discord según sus propios términos de servicio.

9. Contacto

Para cualquier consulta sobre privacidad, abre un issue en GitHub.
